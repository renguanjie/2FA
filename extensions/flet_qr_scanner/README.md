# flet-qr-scanner
FletQrScanner control for Flet

## Installation

Add dependency to `pyproject.toml` of your Flet app:

* **Git dependency**

Link to git repository:

```
dependencies = [
  "flet-qr-scanner @ git+https://github.com/MyGithubAccount/flet-qr-scanner",
  "flet>=0.85.3",
]
```

* **uv/pip dependency**

If the package is published on pypi.org:

```
dependencies = [
  "flet-qr-scanner",
  "flet>=0.85.3",
]
```

Build your app:
```
flet build macos -v
```
